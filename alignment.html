Calls the DTW alignment function and refines the results with the HMM alignment algorithm, with both a basic and modified state spaces (based on the lyrics).

Gets values for DTW alignment and YIN analysis of specified audio signal and MIDI file

runDTWAlignment(audiofile, midorig, tres, ...)

Perform a dynamic time warping alignment between specified audio and MIDI files.

align_midi_wav(MF, WF, TH, ST, width, tsr, ...)

Align a midi file to a wav file using the "peak structure distance" of Orio et al. that use the MIDI notes to build a mask that is compared against harmonics in the audio.

alignment_visualiser(trace, mid, spec[, fig])

Plots a gross DTW alignment overlaid with the fine alignment resulting from the HMM aligner on the output of YIN.
